Mantenimiento Pro: Limpieza y Lubricación de Cadena

Materiales Necesarios

  • Desengrasante biodegradable.
  • Lubricante para cadena (seco o húmedo).
  • Cepillo de cerdas duras o limpiador de cadenas.
  • Trapo de microfibra limpio.
  • Soporte para bicicleta (opcional).

Pasos a Seguir

  1. Desengrasar: Aplica el desengrasante mientras giras los pedales hacia atrás.
  2. Cepillar: Usa el cepillo para remover el lodo entre los eslabones.
  3. Secar: Pasa el trapo por la cadena hasta que no suelte residuos negros.
  4. Lubricar: Aplica una gota de aceite en cada eslabón.
  5. Retirar exceso: Limpia el exceso de aceite con el trapo para evitar atraer polvo.
